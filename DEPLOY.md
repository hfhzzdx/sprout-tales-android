# 安装与使用指南

设备要求
- Android 8.0+（minSdk 26），推荐 Android 12～15。

首次安装
1. 从 GitHub Actions 的构建工件（Artifacts）下载 app-release.apk
2. 在手机设置中允许“安装未知应用”
3. 点击安装包完成安装

首次运行建议
- 打开手机 设置 → 辅助功能/语言 → 文字转语音（TTS），选择你的 TTS 引擎（如“谷歌文字转语音”），并下载中文离线语音包
- 在应用家长区设置语速/音调

导入故事
- 支持 .txt 与 .epub
- 在应用家长区点击“导入故事”，选择文件即可
- 外部故事不会联网上传，仅存储在应用专属目录

后台播放与定时
- 支持锁屏/通知栏控制
- 支持 10/20/30/60 分钟定时播放结束

隐私
- 应用不申请网络权限，不采集任何数据。
